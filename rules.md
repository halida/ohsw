# 《One-Hour Skirmish Wargames》核心规则精简版

## 一、基本概念

**模型**：每个模型代表一个士兵/车辆，单个模型就是独立作战单位。
**牌**：每人一副 52 张扑克 + 2 张 Joker，A=1，J=11，Q=12，K=13，同点比花色：黑桃 > 红桃 > 方块 > 梅花。
**游戏区域**：建议 3-4 英尺（90-120cm）见方，多放地形提供掩护。

## 二、回合流程

```
1. **比先手**：各翻一张牌，大牌先走
2. **交替玩家阶段**：玩家轮流行动，直到有人翻出 Joker，回合立即结束
3. **士气检定**：双方同时检查军队是否崩溃
4. **伤亡检定**：每个倒地模型抽牌，红牌阵亡移除，黑牌复活站起来
5. **开始下一回合**
```

## 三、玩家阶段

1. 翻一张牌，获得 **行动点数(AP)** = 牌面值
2. 激活模型，消耗 AP 进行移动和/或射击：
   - 第一次移动：**1 AP**
   - 第二次移动：**3 AP**
   - 第三次移动：**5 AP**
   - **规则**：可以移动多次再射击，但射击后不能再移动。每个模型每阶段只能射击一次，只能激活一次。
3. AP 用完或选择停止，阶段结束，超额 AP 丢失，不能保留。

## 四、移动

**步兵**：6 英寸（约 15cm）
**骑兵/武士**：9 英寸
**轮式车辆**：公路 24 英寸，越野 6 英寸
**履带车辆**：12 英寸
****特殊**：越过高肩障碍/进入建筑算作一次移动（消耗 1AP），攀爬高障碍物算作两次移动。

## 五、近战

- 模型移动到**接触敌人模型**立即触发
- **攻击者**抽 2 张留大，**防御者**抽 1 张
- 输方**直接阵亡移除**，不是击倒
- 已经倒地被近战直接阵亡，无需检定
- 翻出 Joker 则近战取消，攻击者退 2 英寸，回合结束

## 六、射击

**步骤**：
1. 检查射程和视线，花费 1AP 开火
2. 攻防各抽一张判定牌比大小
3. 攻赢 = 目标**倒地**（侧放模型）
4. 多发武器：第一发没中可以继续抽牌比
5. 目标倒地后剩余子弹可以转火 3 英寸内任意目标，不需要视线（流弹/跳弹）

**防御抽牌加成**（地形掩护）：
- 软掩护（草丛、树后）：**抽 2 张留大**
- 硬掩护（碎石、墙后）：**抽 3 张留大**
- 碉堡：**抽 4 张留大**

**视线规则**：超过模型一半高的地形阻挡视线。贴著肩高线性障碍可以对射。

| 重坦克 | 6/3/2 |

### 射击车辆流程

1. 检查穿甲能力——只有穿甲武器能打装甲车辆
2. 各抽一张判定是否击中
3. 击中再各抽一张：
   - 攻击者 + 武器穿甲值
   - 防御者 + 对应方向装甲值 + 车辆士气
4. 差值就是累计伤害
5. **立即检定乘员士气**：抽一张牌，如果**牌值 ≤ 累计伤害**，乘员弃车，车辆击毁，乘员算伤亡

**重要**：步兵贴脸能近战攻击车辆，自动获得穿甲 1。

## 十、士气与胜负

**崩溃检定**（回合结束进行）：
1. 计算累计**阵亡**数量（不含倒地）
2. 抽一张判定牌 + 军队**士气值**
3. 如果 `(牌 + 士气) > 阵亡数` = 检定通过，继续战斗
4. 如果 `(牌 + 士气) ≤ 阵亡数` = 军队崩溃，你输

- **指挥官加成**：能多抽牌，例如指挥官(2) 抽 1+2=3 张选最喜欢的。只有站着的指挥官能用，每回合只能用一个。

**伤亡检定**（崩溃检定后进行）：
- 每个倒地模型抽牌
- **红牌（红桃/方块）** = 阵亡移除
- **黑牌（黑桃/梅花）** = 站起来继续战斗

**胜利条件**：
- 对方军队先崩溃
- 完成剧本特定目标
