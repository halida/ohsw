# 第二次世界大战 (1939 ~ 1945) 规则精简

## 特殊规则

### 步兵武器

| 武器 | 参数 |
|------|------|
| **突击步枪** | 1 发无限射程 **或** 2 发 18 英寸射程 |
| **西蒙诺夫反坦克步枪** | 射程 18 英寸，1 发，穿甲 2 |
| **巴祖卡/PIAT/铁拳/坦克杀手** | 射程 18 英寸，1 发，穿甲 4。打步兵最小射程 10 英寸，爆炸半径 2 英寸 |

### 坦克/装甲总结

| 车辆类型 | 装甲 前/侧/后 |
|----------|--------------|
| 卡车/吉普 | 0/0/0 |
| 半履带/装甲车 | 1/1/1 |
| 轻坦克 | 2/1/1 |
| 中坦克（谢尔曼） | 4/2/1 |
| 重坦克（虎式） | 6/3/2 |

### 坦克炮穿甲

| 口径 | 穿甲 | 爆炸半径 |
|------|------|----------|
| 2磅炮 | 3 | - |
| 6磅/75mm | 5 | 6磅 2" / 75mm 3" |
| 长管 75mm/77mm | 7 | 3" |
| 17磅/88mm | 9 | 3" |

### 可选军队
- 美国：步兵/空降/海军陆战队/装甲/游骑兵/OSS
- 英国：步兵/空降/皇家海军陆战队/装甲/钦迪特/远程沙漠群/SAS/SOE/国民自卫队
- 苏联：步兵/红军/海军/内务人民委员部/游击队
- 法国：步兵/骑兵/维希/自由法国/摩洛哥人/抵抗组织
- 德国：步兵/装甲师/空降/党卫军/人民冲锋队/勃兰登堡/狼人
- 意大利：沙漠/东线/黑衫军
- 日本：帝国陆军/海军陆战队
