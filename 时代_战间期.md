# 战间期 (1918 ~ 1939) 规则精简

## 特殊规则

### 步兵武器

| 武器 | 参数 |
|------|------|
| **反坦克步枪** | 射程 18"，1 发，穿甲 1 |
| **手榴弹** | 射程 9"，1 发，爆炸半径 2"，穿甲 1。只抽一张命中牌，每阶段只能开枪**或**投弹一次 |
| **三脚架重机枪** | 射程无限，3 发，班组操作 |

### 炮兵

高爆弹规则调整：
- 射击只需要 1 AP
- 判定命中和击倒和单发武器一样，但抽**两张**判定牌不是一张
- 如果命中击倒，对原目标爆炸半径内所有其他模型逐一检定，每个模型抽一张
- 爆炸半径从目标底座边缘量起，碰到底座就影响

| 火炮 | 参数 |
|------|------|
| **迫击炮** | 射程无限，最小射程 10"，1 发，爆炸半径 2"，穿甲 1 |
| **野炮** | 射程无限，最小射程 20"，1 发，爆炸半径 3"，穿甲 3 |

### 火焰喷射器

- **射程**：6"，目标模型自动击倒，然后 3 英寸内每个模型抽一张，防御者不管地形都只能抽一张防御
- 对车辆无效
- 携带者因为沉重只能移动 4"

### 车辆

**通用**：
- 车辆士气和军队相同，是军队一部分
- 轮式：越野 6" / 公路 24"。履带：12"
- 只能沿道路移动（剧本限制），不能过障碍物
- 伤害累计，直到乘员弃车。弃车后留在场上当作地形，乘员算伤亡

**射击车辆流程**：
1. 各抽一张判定是否击中
2. 击中再各抽一张：攻击 + 穿甲，防御 + 装甲 + 士气
3. 差值就是累计伤害
4. 立即检定：抽一张，如果牌 ≤ 累计伤害，弃车

**装甲**：
```
无装甲/马车: 0/0/0
装甲车/一战坦克: 1/1/1
轻坦克: 2/1/1
```

**穿甲**：
```
反坦克步枪: 1
重机枪穿甲弹: 1，跳弹
巴祖卡/RPG/PIAT: 4
机关炮: 2
轻坦克炮: 3
中坦克炮: 5
```

## 参考配兵（剧本 3：自由军团）

**自由军团 (35 分)** 士气 3
```
- Kapitän zur Flieg von Krumen，手枪，领导力(4)
- Countess Ludmilla，手枪，领导力(1)
- Crazy Adolph，冲锋枪，手榴弹，手榴弹专家（命中抽两张）
- 2 x 突击工兵，冲锋枪，手榴弹
- 1 x 突击工兵，刘易斯机枪
- 1 x 突击工兵，反坦克步枪
- 8 x 突击工兵，步枪
```

**布尔什维克 (30 分)** 士气 2
```
- Captain Yudenich，军事专家，领导力(2)，手枪
- Commisar Krivitsky，手枪
- 10 x 布尔什维克，步枪
- 1 x 马克沁机枪 带班组
- 装甲车：装甲 1/1/1，2 乘员，1 x 重机枪
```
