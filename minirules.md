# One-Hour Skirmish 一分钟规则
**(打印一页版)**

---

### 牌值
A=1, J=11, Q=12, K=13
**同点数比花色：黑桃 > 红桃 > 方块 > 梅花**
完全相同牌（同点同花）→ 双方各重抽一张

### 回合
```
1. 比先手 → 各抽一张，大牌先走
2. 交替玩家阶段 → 直到有人翻出 Joker，回合立即结束
3. 士气检定 → 双方同时检查
4. 伤亡检定 → 倒地模型抽牌，红=死，黑=活
5. 下一回合
```

### 玩家阶段
- 翻一张牌 = 获得 **AP = 牌面值**
- 激活模型：`移动 1次 = 1AP，移动 2次 = 3AP，移动 3次 = 5AP`
- 射击 1AP，**射击后不能移动**，移动后可以射击。每阶段只能射击一次，激活一次。

### 移动
```
步兵 ................ 6"
骑兵/武士 .......... 9"
轮式车辆 ..... 公路 24" / 越野 6"
履带车辆 ............ 12"
```

### 近战
- 模型移动到**接触敌人** = 立即近战
- 攻击者抽 2 留大，防御者抽 1 → 输直接死，不是击倒

### 射击
- 攻 1 张 vs 防 1 张 → 攻赢 = 目标倒地
- 多发武器依次射击，目标倒地后剩余子弹可以转火 3" 内任意目标
- 防御抽牌加成（抽多张留最大）：`软掩护+1 硬掩护+2 碉堡+3`
  - 软掩护：植被、灌木丛等
  - 硬掩护：碎石、大树、墙壁等
  - 碉堡：混凝土工事、永备防御

### 射击车辆
```
1. 各抽一张 判断命中
2. 命中再各抽一张：
   攻击者 + 穿甲值
   防御者 + 装甲 + 士气
3. 差值 = 累计伤害
4. 乘员检定：抽一张，牌 ≤ 伤害 → 弃车，击毁
```
- 步兵贴脸近战 = 穿甲 1

| 快速 | 移动 +3" |

### 士气崩溃（回合结束）
- `(抽牌 + 军队士气) > 累计阵亡` → 通过
- `(抽牌 + 军队士气) ≤ 累计阵亡` → 崩溃，你输
- 指挥官可以多抽牌，选最好的

### 胜利
- 对方先崩溃
- 完成剧本目标
